Buddies go
==========

A space to learn Go in an ad-hoc manner.

Pre-requisites
--------------

-   Go compiler \>= 1.20
-   `golangci-lint`
-   `goimports`

See guidance on [installing tools](./doc/installing-tools.md) if you
don't have a preferred way already.

Building
--------

    go run print.go
